# gamercrud-dragonheir

## Overview

This is a simple Data centered template designed to be used to quickly get you up and running 
in a data based site.


## Geeking
See [Developer](Developer.md) document for basic how-to and [Ideas](Tech.md) for tech and general roadmaps. 

## Acknowlegements

The general scaffolding is implemented using the [Quarkus Renarde TODO demo app](https://github.com/FroMage/quarkus-renarde-todo) for out of the box User management (OIDC / email login), CRUD Database entities, i18n [Language support](https://www.w3.org/International/i18n-drafts/nav/about) and basic front end website.  The original (and much appreciated) TODO app may be removed in the near future.
